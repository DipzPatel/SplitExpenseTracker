# Split Expense Tracker

A modern Android application to split expenses among friends and family.

## Features

- Create Groups
- Add/Edit/Delete Members
- Add/Edit/Delete Expenses
- Multiple Expense Participants
- Automatic Balance Calculation
- Settlement Suggestions
- Light & Dark Theme
- Material Design 3 UI

## Tech Stack

- Kotlin
- MVVM Architecture
- Room Database
- Coroutines
- Flow
- RecyclerView
- Material Design 3
